PART C â€” ROUTING: Connect URL â†’ View â†’ Template
STEP 8 â€” Add URL for the blog app

-- blog/urls.py -- (create file if missing)

`from django.urls import path

    path('', views.post_list),
]

Meaning:
When user goes to /blog/, call post_list() in views.py.


STEP 9 â€” Include blog routes in main project

    path('admin/', admin.site.urls),

]


Meaning:

anything under /blog/ will be routed to blog app.

PART D â€” VIEW: Fetch data & send to template
STEP 10 â€” Write the view

-- blog/views.py --

`from django.shortcuts import render
from .models import Post

def post_list(request):
    posts = Post.objects.all()
    return render(request, 'blog/post_list.html', {'posts': posts})`


Meaning:
Fetch all posts from DB using ORM
Return HTML template
Pass variable posts

Meaning:
Loop through posts
Display fields
